利用 Gitpage 和 Hexo 搭建免费个人 Blog

发表于 2019-09-14 更新于 2019-09-15

前言

试用了不少博客系统，还是喜欢 Hexo 这种简洁风格，单纯记录文字的系统，此篇就用来记录我搭建 Hexo Blog 的过程吧。

先对 Hexo 介绍一下，Hexo 是一个基于node.js制作的快速、简洁且高效的博客框架，与传统的开源博客系统不同，其实 Hexo 就是一个静态页面生成系统，根据你的配置生成相应的页面，然后上传部署到你的网站空间，且支持 Markdown 很适合只想专注于文章的用户。

主页： Hexo中文主页

前期准备

前面我们知道了 Hexo 基于的是 node.js 那么运行环境你是不得不装的，还有一个必备的东西是 Git。安装过程这里不赘述，没有这两个环境的小伙伴请到 Hexo 官网文档主页概述部分进行查看。

当然你可以自己租用服务器存放你的 Blog 页面，本文主要介绍利用 Github 的 Page 功能，搭建我们的免费博客。既然要用到 Github 还不赶紧去注册一个账号嘛 lol 传送门。
